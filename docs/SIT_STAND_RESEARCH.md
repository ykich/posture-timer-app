# 座る・立つのバランスについて

専門機関や大学の研究に基づいた、座りすぎ・立ちすぎの目安と理想的なバランスをまとめています。

## 座りすぎのリスク

- **連続して座る時間の目安：30分〜1時間**
- 30分以上連続して座り続けると、下半身の血流が滞り、代謝の低下や腰痛、将来的な健康リスクにつながります
- 1日の合計時間よりも「こまめに中断すること」が重要です

日本の健康ガイドラインでは「30分に1回、少なくとも1時間に1回は立ち上がる」ことが強く推奨されています。

## 立ちすぎのリスク

- **連続して立つ時間の目安：30分〜45分**
- 長時間の立位は、足のむくみや疲労、腰への局所的な負担を増大させます

カナダのウォータールー大学（生体力学・人間工学）の研究では、連続して2時間立ち続けると、腰痛のない健康な人でも約半数が腰痛を発症することが確認されています。

## 理想的な「座る・立つ」の比率

| 提唱機関・研究 | 理想の割合・ルール | 1時間あたりの配分例 |
| :--- | :--- | :--- |
| **ウォータールー大学** | 座る：立つ = 1:1 〜 1:3 | 座る：15〜30分 / 立つ：30〜45分 |
| **コーネル大学**（エルゴノミクス） | 「20-8-2」の法則 | 座る：40分 / 立つ：16分 / 動く：4分 |

「座り」と「立ち」をこまめに切り替えること（早めに、頻繁に動くこと）が最も重要とされています。どちらの姿勢も「長時間同じ状態を続けること」が体に最も負担をかけます。まずは**30分ごとに姿勢を変える（または少し歩く）**ことを習慣にしましょう。
